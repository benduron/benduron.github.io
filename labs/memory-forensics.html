Memory Forensics Lab

Memory analysis using Volatility to detect malware and reconstruct attacker activity.

Goal

Analyze a memory image to identify malicious processes/artifacts and infer attacker behavior from in-memory evidence.

Tools & Techniques

  • Volatility Framework (process trees, network artifacts, extracted indicators)
  • Host-based evidence correlation (process + connections + persistence clues)

What I Did

  • Reviewed process hierarchy to identify suspicious/unknown processes.
  • Checked memory-resident network indicators and supporting artifacts.
  • Documented findings into an incident-style narrative and next steps.
