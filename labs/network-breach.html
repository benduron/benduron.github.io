Network Breach Analysis Lab

Simulated incident response investigation of a compromised Windows host.

Goal

Investigate a suspected compromise by identifying indicators of compromise (IOCs), suspicious processes, and evidence of attacker activity.

Tools & Techniques

  • Host investigation (process review, artifact review)
  • Network verification (connections/communications)
  • Documentation and escalation-style reporting

What I Did

  • Collected evidence from the host and reviewed suspicious execution artifacts.
  • Identified potential IOCs and abnormal behavior consistent with compromise.
  • Validated findings using network/process context and documented conclusions.
