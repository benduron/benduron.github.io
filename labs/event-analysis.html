Event Analysis Lab

Packet-level investigation using Wireshark and NetworkMiner to identify anomalous activity.

Goal

Analyze packet capture data (HTTP/DNS/DHCP/FTP) to identify suspicious behavior and build an evidence-based incident narrative.

What I Did

  • Inspected protocol distribution and traffic patterns to establish a baseline.
  • Reviewed key conversations/streams to identify anomalous activity.
  • Used multiple tools to extract context (hosts, sessions, transferred content) and documented conclusions.
