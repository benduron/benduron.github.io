Dog Daycare Management App (JavaFX)

Custom scheduling and pricing system built in Java for a real small business.

Project Overview

Designed and built a JavaFX desktop application to help manage dog profiles, bookings, scheduling, and pricing for a small dog boarding business. The application handles user input validation, dynamic pricing calculations, and real-time schedule filtering.

Key Features

  • Multi-tab GUI interface (Dogs, Bookings, Schedule)
  • Object-oriented design (Dogs & Bookings classes)
  • Tier-based pricing logic based on dog weight
  • Date-based booking calculations using ChronoUnit
  • Domain join-style validation and error handling
  • Live schedule filtering by selected date
  • Centralized alert/error handling system

Technical Concepts Demonstrated

  • JavaFX UI design (TabPane, VBox, HBox, ListView, DatePicker)
  • OOP design principles
  • ArrayList data structures
  • Input validation & exception handling
  • Business logic implementation
