Windows Server & Active Directory Home Lab

Self-directed infrastructure lab built using Oracle VirtualBox.

Objective

Build a virtualized enterprise-style environment to practice domain administration, user management, and network troubleshooting.

Environment Built

  • Oracle VirtualBox virtualization platform
  • Windows Server 2016 (Domain Controller)
  • Active Directory Domain Services (AD DS)
  • Custom domain: benduron.com
  • Windows 10 client machine joined to domain

Key Implementations

  • Installed and promoted Windows Server to Domain Controller
  • Configured DNS and domain services
  • Created and managed user accounts and group memberships
  • Implemented helpdesk-style restricted permissions
  • Configured static IP addressing within virtual network
  • Installed RSAT tools for remote administration
  • Practiced command-line troubleshooting (ipconfig, net user, net use)

Skills Developed

  • Active Directory administration
  • Domain join & trust troubleshooting
  • User access control
  • Network configuration
  • Virtualization fundamentals
  • Windows server management
